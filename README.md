Intelligent Machine Vision Platform

This project is a computer vision platform that integrates multiple AI modules for different real-world applications.

->Modules

- **defect_detection**  
  Detects defects in industrial products using computer vision.

- **epp_detection**  
  Detects if people are wearing personal protective equipment (helmet, vest, etc.).

- **face_search**  
  Performs face detection and recognition.

- **parking**  
  Smart parking system for vehicle detection and space monitoring.

- **posture**  
  Human posture detection and analysis.

- **main.py**  
  Main file that integrates all modules.

Technologies
